# pytt

pytt(PYThagorean Triplets) is a simple tool that outputs pythagorean triplets
in csv form.

## Planned features

- [ ] Support different generators.([Formulas])

# Installation

```
go install github.com/cpmachado/pytt@latest
```

## Usage

```
Usage of pytt:
  -k int
    	number of triplets (default 100)
  -n int
    	upper bound (default 9223372036854775807)
  -t	print header
  -v	display version
```
