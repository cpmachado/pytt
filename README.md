# pytt

pytt(PYThagorean Triplets) is a simple tool that outputs pythagorean triplets
in csv form.

## Planned features

- [ ] Support different generators.([Formulas])

# Installation

```
go install github.com/cpmachado/pytt@latest
```

## Usage

```
pytt is a simple tool to compute pythagorean triplets, in csv form.

Usage of pytt:
  -n int
    	upper bound of each element (default 100)
  -t	print header
  -v	print version
```
